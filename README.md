# tinyredis 

A minimal implementation of Redis.

### Features
- [x] PING
- [x] ECHO
- [x] GET
- [x] SET
- [x] Cache with Expiry
- [ ] Transations: multiple operations
- [ ] Pub/sub
- [ ] Backups
- [ ] Leader elections

### Usage and Testing

Use `cargo run` to run the server. Use `cargo test` to run the tests. Both should be done separetely in two different shells.
